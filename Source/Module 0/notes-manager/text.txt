Устанавливаем

Что такое Node.js: Среда выполнения JavaScript вне браузера,
позволяющая запускать JavaScript-код на сервере или локальном компьютере.

Для запуска JavaScript-кода в терминале и разработки серверных приложений.

Устанавливаем ноду

Откройте терминал или командную строку.
Введите команду node -v — должна отобразиться версия Node.js.
Введите команду npm -v — должна отобразиться версия npm (Node Package Manager).
